# **챕터1 시작하기**

---

## **파이썬 프롬프트**
```python
>>> print("Hello Python interpreter!")
Hello Python interpreter!
```
###### '>>>'는 파이썬 프롬프트라고 부릅니다. <br>터미널에서 사용하며,<br> `ctrl`+`D` 혹은 `exit()`명령어로 파이썬 프롬프트를 종료합니다.

---
